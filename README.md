# Cappacita Dev | by Smarkio 🚀️

## Desafio final do módulo 5 com Create React App

Este projeto foi inicializado com o [Create React App](https://github.com/facebook/create-react-app). 

## Sobre o projeto ☕💻
O projeto Cappacita Movie foi desenvolvido durante o curso de Programação para IAs Conversacionais. O desenvolvimento do projeto foi recriando a interface do Netflix, as imagens dos filmes foram utilizando a API do site [The Movie Database - TMDB](https://www.themoviedb.org/).
